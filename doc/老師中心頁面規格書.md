## 1. 全局元件 (Global Components)

### 1.1 Header Section (頁首)

- **目的**: 提供在會員中心的全局導航，確保使用者可以輕鬆返回網站前台主要區域。
- **元件**:
    - **Logo**: 點擊返回首頁。
    - **導航連結**:
        - **首頁 (Home)**
        - **課程總覽 (CourseList)**
        - **一對一諮詢 (ConsultCTA)**
        - **資源分享 (ResourceShare)**
    - **功能按鈕**:
        - **購物車 (Cart)**
        - **登出 (Logout)**: 結束登入狀態，返回首頁。

### 1.2 Nav Section (側邊導航欄)

- **目的**: 作為會員中心內部各功能頁面的主要入口。
- **結構**:
    - 這應是一個垂直的側邊導航欄，讓使用者清晰地看到所有可用的管理選項。
    - 某些項目應根據使用者身份（學生/老師）動態顯示。
- **導航項目**:
    - **購課記錄 (PurchaseRecord)**: (學生) 查看所有課程購買與退款記錄。
    - **我的課程 (MyCourse)**: (學生) 進入上課頁面。
    - **課程上架 (CourseUpload)**: (老師) 建立與編輯課程內容。
    - **統計/分析 (Statistics)**: (老師) 查看課程銷售與學生互動數據。
    - **個人簡介 (Profile)**: (共用) 編輯個人公開頁面資訊。
    - **老師聯絡簿 (TeacherContact)**: (老師) 批改作業、回覆學生問題。
    - **設定 (Settings)**: (共用) 帳號相關設定。
    - **一對一諮詢 (Consultation)**: (共用) 管理預約或提供諮詢服務。
    - **資源分享 (Resource)**: (共用/老師) 撰寫與分享文章。

## 2. 頁面規格 (Page Specifications)

### 2.1 購課記錄 (Purchase Record)

- **身份**: 學生
- **目的**: 提供清晰的消費歷史記錄。
- **元件**:
    - **篩選器 (FilterSection)**:
        - **消費明細**: 正常購買記錄。
        - **退款交易明細**: 退款成功記錄。
        - **平台交易明細**: (可選) 與平台點數或優惠相關的記錄。
    - **列表**: 表格形式顯示每筆交易的日期、課程名稱、金額、狀態。

### 2.2 我的課程 (My Course)

- **身份**: 學生
- **目的**: 學生學習的主要入口，包含其購買的所有課程，並從這裡進入上課頁面。
- **元件**:
    - **課程列 (CourseListSection)**:
        - 以課程卡片(CourseCard)陳列所有已購課程。
        - 卡片內容包含：課程圖片、標題、簡介及**課程進度條**。
        - 點擊卡片後進入**上課頁面 (CoursePageSection)**。
    - **上課頁面 (CoursePageSection)**:
        - **影片播放區 (VideoPlaySection)**:
            - **影片撥放器 (VideoPlayer)**: 支援速度調整、畫質切換、字幕等功能。
            - **影片討論串 (VideoDiscussion)**: 在影片時間軸上發問與討論。
        - **影片下區域 (VideoBelowSection)**:
            - **筆記 (NotesSection)**: 提供個人筆記、觀看共筆、AI摘要功能。
            - **聯絡簿 (ContactBookSection)**: 查看老師指派的作業、測驗，並接收訊息。
        - **章節列表 (ChapterListSection)**:
            - 顯示所有課程章節，並標示目前觀看進度。
            - 點擊可切換影片。

### 2.3 課程上架 (Course Upload)

- **身份**: 老師
- **目的**: 提供一個流暢的介面，讓老師可以建立、編輯並發布課程。
- **元件**:
    - **課程大綱編輯 (OutlineEdit)**:
        - 填寫課程表單(CourseForm)，包含課程名稱、簡介、封面圖、目標受眾等。
    - **影片上傳區 (VideoUpload)**:
        - 建立章節與單元(ChapterList)。
        - 上傳各單元的教學影片(VideoPlayer)。
    - **定價編輯區 (PriceEdit)**: 設定課程售價與優惠方案。
    - **送出審核區 (SubmitReview)**: 完成所有設定後，提交給平台進行審核。

### 2.4 統計/分析 (Statistics)

- **身份**: 老師
- **目的**: 將數據視覺化，幫助老師了解課程銷售狀況與學生學習成效。
- **元件**:
    - **儀錶板 (Dashboard)**:
        - **本月購課折線圖 (MonthlyChart)**: 顯示近期銷售趨勢。
        - **上架課程購課人數統計表 (CourseStats)**: 顯示各課程的總銷售量與收入。
    - **已上架課程 (UploadedCourses)**:
        - 列表顯示所有已上架的課程。
        - 提供「更新課程」按鈕，導向課程編輯頁面。
    - **課程上架 (CourseUploadLink)**: 快速前往課程上架頁面的捷徑。

### 2.5 個人簡介 (Profile)

- **身份**: 共用
- **目的**: 讓使用者（特別是老師）編輯自己的公開檔案，建立專業形象。
- **元件**:
    - **個人簡介表單 (ProfileForm)**:
        - 編輯個人資訊，如頭像、姓名、專業領域、經歷、社群連結等。
        - 提供「發布」與「預覽」功能。
    - **我的證書 (MyCertificates)**: 上傳並展示專業證照。
    - **我的作品 (MyWorks)**: 陳列個人作品集或專案連結。

### 2.6 老師聯絡簿 (Teacher Contact)

- **身份**: 老師
- **目的**: 老師與學生互動的核心功能，包含批改作業與回覆問題。
- **元件**:
    - **已上架課程 (UploadedCourses)**: 選擇要管理的課程。
    - **改聯絡簿 (ContactBookEdit)**:
        - 進入類似學生上課的介面，但功能為管理與回覆。
        - **影片討論串**: 回覆學生在影片時間軸上的提問。
        - **聯絡簿**: 批改學生提交的作業、發布測驗、傳送訊息。

### 2.7 設定 (Settings)

- **身份**: 共用
- **目的**: 進行帳號層級的設定。
- **元件**:
    - **一對一諮詢功能開啟 (ConsultToggle)**: (老師) 老師可選擇是否要開放自己的諮詢服務。
    - **Nav 編輯 (NavEdit)**: 讓使用者可以自訂側邊導航欄的項目順序或顯示與否。

### 2.8 一對一諮詢 (Consultation)

- **身份**: 共用
- **目的**: 學生預約諮詢，老師管理諮詢時段與進行會談的介面。
- **元件**:
    - **行事曆 (Calendar)**:
        - 老師設定可預約的時段。
        - 學生查看並預約/取消時段。
    - **訪談室 (InterviewRoom)**:
        - 時間到時進入的線上會議室。
        - **視訊框 (VideoFrame)**: 雙方視訊畫面。
        - **對話框 (ChatBox)**: 文字訊息、檔案傳輸。
        - **計時器 (Timer)**: 顯示剩餘時間並進行計費(Billing)。

### 2.9 資源分享 (Resource)

- **身份**: 共用/老師
- **目的**: 讓老師或被授權的使用者發布專業文章。
- **元件**:
    - **文章撰寫 (ArticleWrite)**:
        - 提供一個 rich text editor 進行文章編輯。
        - **文章分類 (ArticleCategory)**: 選擇文章所屬的分類 (技術新訊、知識分享等)。
        - **文章發布 (ArticlePublish)**: 提交文章，等待審核或直接發布。
    - **筆記分享 (NoteShare)**: (可選) 學生可選擇是否要將自己的課程筆記公開分享。
